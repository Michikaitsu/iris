Usage Guide

Master IRIS with this comprehensive guide

Basic Usage

1. Launch the Web Interface

python src/start.py web

Open your browser to http://localhost:8000

2. Write Your Prompt

Describe what you want to see:

"a detailed portrait of a cyberpunk warrior, neon lighting, highly detailed, masterpiece, 8k"

3. Adjust Parameters

  • Steps: 20-30 for most images
  • CFG Scale: 8-12 for balanced results
  • Resolution: Start with 512×512

4. Generate!

Click "Generate" and wait for your masterpiece.

Parameters Explained

CFG Scale (Guidance)

Controls how closely the AI follows your prompt.

  • Low (5-7): More creative, less accurate
  • Medium (8-12): Balanced (recommended)
  • High (15+): Very literal, may oversaturate

Sampling Steps

Number of refinement iterations.

  • 20 steps: Fast, good quality
  • 30-50 steps: Better quality
  • 50+ steps: Diminishing returns

Prompt Engineering

Master the art of writing effective prompts

Prompt Structure

Basic Template:
[Subject] + [Style] + [Quality] + [Details]

1. Subject (What)

The main focus: "a warrior", "mountain landscape", "futuristic city"

2. Style (How it looks)

Art style: "digital art", "oil painting", "photorealistic", "anime"

3. Quality Tags

Modifiers: "masterpiece", "highly detailed", "8k", "professional"

Effective Examples

Portrait Photography

"close-up portrait of a woman, studio lighting, bokeh background, professional photography, highly detailed, 8k, photorealistic"

Good for: Professional headshots, character portraits

Cyberpunk Scene

"cyberpunk city street at night, neon lights, rain puddles reflecting lights, blade runner style, cinematic composition"

Good for: Sci-fi environments, atmospheric scenes

Negative Prompts

Tell the AI what you DON'T want. Essential for quality results.

Common Negative Prompt:
"low quality, worst quality, blurry, distorted, ugly, deformed, disfigured, bad anatomy, wrong anatomy, extra limbs, watermark, text, error"

Artifact Gallery

When AI gets a little too creative.
Common artifacts & how to deal with them

What Are Artifacts?

Artifacts are visual errors caused by conflicting data, resolution limits or prompt ambiguity.

The Clone Saga

Symptom: Two (or more) heads, merged faces

Cause: High resolution (1024px+)

Fix: Use 512×768 and upscale

Hand Horror

Symptom: 6+ fingers, backwards thumbs

Cause: Confusing hand pose training data

Fix: Add normal hands, five fingers to prompt

Almost-Words

Symptom: Text like "ČÅFĒ" instead of "CAFE"

Cause: Stable Diffusion is bad at text

Fix: Add text in post-processing or use SDXL

Prevention Tips

  1. Start small: 512×512 → upscale
  2. Strong negatives
  3. Simple prompts beat complex ones
  4. Test different seeds
  5. CFG sweet spot: 8–12 (not 20)

FAQ

Frequently Asked Questions about IRIS

IRIS (Intelligent Rendering & Image Synthesis) is a Stable Diffusion-based image generation system.

Minimum: GTX 1650 with 4GB VRAM

Recommended: RTX 3060 with 12GB VRAM

High-End: RTX 4090 with 24GB VRAM

Start with a clear subject, add style tags, and include quality descriptors.

Example: "a detailed portrait of a cyberpunk warrior, neon lighting, masterpiece, 8k"

Solutions:

  1. Update NVIDIA drivers
  2. Reinstall PyTorch with matching CUDA version
  3. Verify GPU compatibility
  • Reduce image resolution (try 512x512)
  • Lower batch size to 1
  • Enable DRAM extension
  • Close other GPU applications

Troubleshooting

Solutions to common problems

Quick Fixes

Out of Memory

  • Reduce resolution to 512×512
  • Lower batch size to 1
  • Enable DRAM extension
  • Close other applications

CUDA Not Detected

  • Update NVIDIA drivers
  • Reinstall PyTorch + CUDA
  • Verify GPU in Device Manager
  • Check CUDA installation

Slow Generation

  • Update GPU drivers
  • Check GPU utilization
  • Close background processes
  • Verify CPU isn't bottlenecking

Poor Image Quality

  • Adjust CFG scale (8-12)
  • Increase sampling steps
  • Improve prompt quality
  • Add negative prompts

API Reference

Integrate IRIS into your applications

Getting Started

Base URL

http://localhost:8000/api/v1

Authentication

API key required for external access. Include in header:

Authorization: Bearer YOUR_API_KEY

Endpoints

POST /generate

Generate images from text prompts.

Request Body:

{
  "prompt": "a detailed portrait, highly detailed",
  "negative_prompt": "low quality, blurry",
  "steps": 30,
  "cfg_scale": 8,
  "width": 512,
  "height": 512
}

Code Examples

Python

import requests

url = "http://localhost:8000/api/v1/generate"
data = {
    "prompt": "a detailed portrait, highly detailed",
    "steps": 30,
    "cfg_scale": 8
}

response = requests.post(url, json=data)
result = response.json()
