CUDA/GPU Issues

Problem: "CUDA not available" or "No GPU detected"

Your system isn't detecting the GPU for acceleration.

Step 1: Check NVIDIA Drivers

nvidia-smi

Should show your GPU. If error, update drivers from nvidia.com

Step 2: Reinstall PyTorch with CUDA

pip uninstall torch torchvision
pip install torch torchvision --index-url https://download.pytorch.org/whl/cu118

Step 3: Verify CUDA in Python

python -c "import torch; print(torch.cuda.is_available())"

Should print "True"

Memory Issues

Problem: "CUDA out of memory" Error

Your GPU doesn't have enough VRAM for the current settings.

Solutions (try in order):

  1. Reduce resolution: 1024‚Üí768‚Üí512
  2. Lower batch size to 1
  3. Reduce sampling steps (50‚Üí30‚Üí20)
  4. Enable low VRAM mode in settings
  5. Close other GPU applications (browsers, games)
  6. Restart IRIS to clear VRAM

Problem: System RAM usage high

Your system is running low on RAM.

  • Close unnecessary applications
  • Increase virtual memory/page file
  • Consider upgrading RAM to 16GB+

Installation Issues

Problem: pip install fails

Dependency installation errors.

# Clear pip cache
pip cache purge

# Upgrade pip
python -m pip install --upgrade pip

# Try install again
pip install -r requirements.txt

Problem: Python version mismatch

IRIS requires Python 3.9-3.11.

python --version

If not 3.9-3.11, install correct version and recreate venv

Problem: Permission errors (Windows)

Installation blocked by permissions.

Run Command Prompt or PowerShell as Administrator, then retry installation.

Server Issues

Problem: Server won't start

Can't access web interface.

  • Check if port 8000 is already in use
  • Try a different port: --port 8080
  • Disable firewall temporarily
  • Check file permissions
  • Run as administrator (Windows)

Problem: Can't access from browser

Server running but browser can't connect.

  1. Verify URL: http://localhost:8000
  2. Try http://127.0.0.1:8000
  3. Check browser console for errors (F12)
  4. Clear browser cache and cookies
  5. Try a different browser

Model Issues

Problem: Model download failed

Can't download or load models.

  • Check internet connection
  • Verify sufficient disk space (5GB+ free)
  • Download manually from HuggingFace
  • Check firewall/antivirus blocking downloads

Problem: Model loading errors

Downloaded model won't load.

  • Verify file integrity (re-download if corrupted)
  • Check file format (.safetensors or .ckpt)
  • Ensure model in correct folder
  • Restart IRIS after adding models
